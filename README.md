# chuvak-calculator

Python3 tkinter calculator with advanced functional!

**Features:**

- Can make simple actions ( like 2 + 2 or 32 - 21 )

- Can calculate factorial ( fac(5) = 120 )

# How to use

**1. Install the archive**

**2. Run in the terminal:**
```bash
python3 calc.py
```

# Dependencies

**1. Tkinter**

Debian/Ubuntu:

```bash
apt install python3-tk
```
Fedora:

```bash
dnf install python3-tkinter
```
